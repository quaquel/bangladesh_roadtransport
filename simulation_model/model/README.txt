Start the bgdapp.jar with the following command, assuming that the parameters.tsv file and folders
like 'infrastructure', 'od', 'gis', 'economics' and 'development' are in the folder E:/BGD:

java -jar bgdapp.jar E:/BGD/


The bgd.jar file is meant for running with the EMA workbench.

The java version should be 8 or higher.
